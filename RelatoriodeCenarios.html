Cenários de Teste - Advantage Online Shopping

1. Cenário: Buscar um Produto Existente

        Feature: Busca de produtos
        
        Scenario: Usuário busca por um produto existente
            Given que o usuário está na página inicial do site
            When o usuário busca pelo produto "HP Pavilion 15z Laptop"
            Then o sistema deve exibir os resultados da busca com o produto "HP Pavilion 15z Laptop"
            And o sistema deve exibir o preço do produto

2. Cenário: Adicionar Produto ao Carrinho

        Feature: Carrinho de compras
        
        Scenario: Adicionar um produto ao carrinho
            Given que o usuário está na página de detalhes do produto "HP Pavilion 15z Laptop"
            When o usuário clica no botão "Adicionar ao carrinho"
            Then o sistema deve adicionar o produto ao carrinho
            And o sistema deve exibir uma notificação de sucesso
            And o carrinho deve exibir a quantidade "1"

3. Cenário: Remover Produto do Carrinho

        Feature: Carrinho de compras
        
        Scenario: Remover um produto do carrinho
            Given que o usuário tem o produto "HP Pavilion 15z Laptop" no carrinho
            When o usuário remove o produto do carrinho
            Then o sistema deve remover o produto do carrinho
            And o carrinho deve estar vazio
            And o sistema deve exibir uma mensagem "Seu carrinho está vazio"

4. Cenário: Registrar Novo Usuário

        Feature: Cadastro de usuário
        
        Scenario: Registrar um novo usuário com sucesso
            Given que o usuário está na página de cadastro
            When o usuário preenche todos os campos obrigatórios com informações válidas
            And o usuário clica no botão "Registrar"
            Then o sistema deve criar a conta do usuário
            And o sistema deve logar o usuário automaticamente
            And o sistema deve redirecionar o usuário para a página inicial

5. Cenário: Login de Usuário Existente

        Feature: Login de usuário
        
        Scenario: Usuário realiza login com sucesso
            Given que o usuário está na página de login
            When o usuário preenche o nome de usuário "usuario_teste" e senha "senha123"
            And o usuário clica no botão "Login"
            Then o sistema deve autenticar o usuário
            And o sistema deve redirecionar o usuário para a página inicial
            And o nome do usuário "usuario_teste" deve ser exibido no cabeçalho

6. Cenário: Atualizar Imagem de Produto

        Feature: Atualizar imagem de produto
        
        Scenario: Atualizar a imagem do produto no carrinho
            Given que o usuário tem um produto no carrinho
            When o usuário acessa a página de detalhes do produto "HP Pavilion 15z Laptop"
            And o usuário altera a cor do produto para "Preto"
            And o usuário clica no botão "Atualizar imagem"
            Then o sistema deve atualizar a imagem do produto no carrinho com a cor selecionada

7. Cenário: Finalizar Compra com Sucesso

        Feature: Finalizar compra
        
        Scenario: Finalizar compra com sucesso
            Given que o usuário tem produtos no carrinho
            And o usuário está logado
            When o usuário acessa a página de checkout
            And o usuário preenche as informações de pagamento e endereço
            And o usuário clica no botão "Finalizar compra"
            Then o sistema deve processar a compra com sucesso
            And o sistema deve exibir uma mensagem de confirmação "Compra realizada com sucesso"
